# SHORTY

Tool to shorten long URL to short one.

## TECH STACK

- Golang
- React
- Mongo
- Echo (Golang Web Framework)


### Feature

- URL Shortener
- Login Auth via JWT
- Custom URL for logged in user
- URL Click analytics
