# Shorty UI

## Getting Started

* Browse the Repo from cmd/termnial and type
    ```
    >npm install or yarn install
    ```
* Build and Start development server
    ```
    >npm start
    ```
    Browse http://localhost:3000 to view project

* Build for Production
    ```
    npm run build
    ```
